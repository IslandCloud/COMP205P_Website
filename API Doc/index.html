Introduction

Using this API, you will be able to receive sets of data, generally in the form of a Key/Value pair. The key will be the GUID of the element, and the Value will be the property (if relevant) associated with that element and type of request. For example, if you request all "Students" and the property that interests you is the "Gender", then you will receive Key/Value pairs like this: Guid: "eb24a5df-515a-42c9-878c-986e2ed214bb" Value: "M"

Set2

The /Set2 set of actions is the second method by which developers can use the API to request Sets.

APIDescription
GET api/Set2?setGuid={setGuid}&setGuid2={setGuid2}&operation={operation}

Request a Set using a known Guid. For example, try this request to get a Set (which is of Type Student with Property Gender): /api/Set2?setGuid=9a7234fd-2e05-4d64-91e5-928aa6d78d4a
GET api/Set2?setType={setType}&setProperty={setProperty}&propertyFilter={propertyFilter}&setProperty2={setProperty2}&propertyFilter2={propertyFilter2}&operation={operation}

The developer can request a Set by providing the Set Type and associated Property required.
