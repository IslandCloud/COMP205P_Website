GET api/Set2?setType={setType}&setProperty={setProperty}&propertyFilter={propertyFilter}&setProperty2={setProperty2}&propertyFilter2={propertyFilter2}&operation={operation}

The developer can request a Set by providing the Set Type and associated Property required.

Request Information

URI Parameters

NameDescriptionTypeAdditional information
setType

The type of Set, for example 'Student', 'School' etc.

 string

Required
setProperty

The property to accompany the set Type. For example, if the Set Type is 'Student', then the property could be 'Gender'. This will tell the API to look for and return the Set of Students (i.e. the Set of their Guids) along with their Gender.
