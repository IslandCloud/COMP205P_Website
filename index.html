Introduction

This whole project is basically about data mining and data visualization, and our job is the first one, build an appropriate data engine that can respond to the graph operations run by the user, and search for the result by the query in millions of data.
The data engine connect to multiple types of data source, and have a REST web API for user to search the data they need in the browser.

Product Features

·  Simple web API, easy and user-friendly to use.
·  API can be used to retrieve set data and graph data.
·  Cloud based storage solution on Microsoft Azure.
·  Automated set generation to generate sets given any database
